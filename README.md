# Trabalho Final - Desenvolvimento Web 1

Este projeto é um website desenvolvido como **Trabalho Final** da disciplina de **Desenvolvimento Web 1**. O objetivo foi criar um portal informativo sobre a história e atualidade da Seleção Brasileira.


## Sobre o Projeto

O site serve como uma fan-page interativa, aplicando conceitos fundamentais de HTMl, CSS e JavaScript.
